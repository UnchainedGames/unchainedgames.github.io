Mythic Titan Raffle FAQ

Answering frequently asked questions all about
Gods Unchained Card Raffles

Are Gods Unchained Card Raffles secure?

The simple answer is yes. All Raffles on the marketplace are self managed by an open source Ethereum smart contract

Checkout the smart contract here: https://etherscan.io/address/0x065ae34771aa010747dbe375dabaf02c0931d778

How do I buy raffle tickets?

1. Visit the marketplace

2. Tap on a card

3. Select the number of raffle tickets

4. Click the Buy Tickets button and Confirm your metamask transaction

As a note the marketplace works best with Google Chrome web browser and a metamask chrome extension

How do I create a raffle?

1. Navigate to Create Raffle

2. Find the card you want to raffle by name and quality

3. Tap on the card

4. Set the number of tickets required to activate the raffle

5. Set the cost per raffle ticket

6. Tap the Create Raffle button and Confirm metamask transaction

How do I see my raffles?

Navigate to My Raffles

Filter by Active, Cancelled, and Completed Raffles

How do I see my raffle tickets?

Navigate to My Tickets

Filter tickets by raffles still Active, raffles that Ended, and your Winning tickets

How do I see raffles I won?

Navigate to My Tickets

Filter tickets by Winner

How do I see which addresses are in a given raffle?

To view participants in a raffle, click the Ticket Icon in the top right of a raffle with tickets

A list of participants can also be queried using The Graph. Simply replace the Raffle ID and hit the Play button

Is there an api for the raffle market?

Yes, the marketplace is powered by The Graph

How do I cancel a Raffle I created?

1. Visit My Raffles

2. Tap the Raffle you want to Cancel

3. Refund all tickets in the raffle

4. Cancel the raffle and your card will be returned to you

How do I withdraw my tickets from an active Raffle?

1. Visit My Tickets

2. Filter by Active

3. Tap the Card

4. Select the number of tickets to withdraw

5. Click Withdraw Tickets and Confirm Transaction

What are No Limit Raffles?

No limit raffles are raffles with no limit to the maximum number of tickets which can be included

Anyone can activate no limit raffles once the minimum number of tickets is reached
